basic git command:

git pull -- aggiornare
git clone https://github.com/Kiwiilock/Project -- ottenere cartella
git push -- upload files
git add git.txt -- add a file to the repository
git checkout -- show differences
git commit -- commits modified files

Standard Workflow
git pull
git add nomefile
git checkout
git commit -m "messaggio commit"
git push
